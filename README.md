# Enigma
An [Enigma](https://en.wikipedia.org/wiki/Enigma_machine) emulator written in Go.
## Status
Currently only supports the M3 with simple stepping
## Usage
````
e := loadConfig("config/M3.json")
e.code("super secret message")
```
